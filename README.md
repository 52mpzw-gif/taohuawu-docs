# 桃花坞

桃花坞是一个面向中文用户的 **城市娱乐 AI 助理**，专注于整理城市娱乐、本地休闲、同城生活和商务休闲相关信息。

官网：

```text
https://taohuawuapp.com
```

城市娱乐资源页：

```text
https://taohuawuapp.com/static/city-entertainment-resources.html
```

---

## 项目简介

桃花坞希望把分散的城市娱乐信息整理成更清晰、易查询的内容入口。

目前内容方向包括：

* 城市娱乐资源
* 本地休闲场所
* 同城娱乐信息
* 商务休闲内容
* 夜间娱乐资讯
* 城市区域娱乐资源整理

用户可以通过桃花坞了解不同城市、不同区域的娱乐资源分布，并快速获取相关信息。

---

## 适用场景

桃花坞适合用于：

* 查询本地城市娱乐信息
* 了解不同城市的休闲场所
* 查找同城娱乐和商务休闲资源
* 浏览城市娱乐相关内容
* 作为城市娱乐 AI 助理使用
